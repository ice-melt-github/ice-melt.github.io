Python自然语言处理勘误笔记

FreqDist对象的inc方法报错

错误信息为:
AttributeError: 'FreqDist' object has no attribute 'inc'

可以修改为:
suffix_fdist[word[-1:]] += 1

FreqDist对象通常像这样使用:
fd = FreqDist(my_text)

对字典的keys()进行切片保存

错误信息为:
TypeError: 'dict_keys' object is not subscriptable

这是因为字典的keys对象不支持切片,转换成list即可

可以修改为: word_feature = list(all_words.keys())[:2000]
